# 🐍 Snake Game in C

A classic Snake Game made in C using Windows Console API.

## Features

- Main Menu
- Difficulty Selection
- High Score Saving
- Golden Fruit
- Poison Fruit
- Pause System
- Sound Effects
- Animated Welcome Screen
- Colored UI

## Controls

W - Up
S - Down
A - Left
D - Right

P - Pause
Q - Quit

## Technologies

- C
- Windows Console API
- File Handling
- Sound API
